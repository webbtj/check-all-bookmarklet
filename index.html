Checkbox (un)Checker

What is it?

Ever been faced with a page full of checkboxes that you need to check/uncheck? Here's a little solution, install this bookmerklet, then, when faced with a pile of checkboxes, click the bookmarklet and type one of the following commands.

  • T - type "t" to toggle all checkboxes on the page (all boxes that are checked will become unchecked, all boxes that are unchecked will become checked).
  • C - type "c" to check all checkboxes on the page (all boxes become checked).
  • U - type "u" to uncheck all checkboxes on the page (all boxes become unchecked).

Commands are case insensitive. Aliases exist: Y(es) and A(ll) are aliases for C(heck); N(o) or N(one) is an alias for U(ncheck).

How does it work?

Add the Checkbox (un)Checker Bookmarklet Button below to your browser's bookmark bar (drag and drop). Then, any time you are faced with multiple checkboxes, click the bookmarklet and type one of the above commands.
